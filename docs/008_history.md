# 008 履歴・記録一覧画面

## 概要
全作業セッションの一覧表示・編集・削除と、手動入力フォームへの導線を持つ履歴画面を実装する。

## 対象フェーズ
Phase 2 — 可視化機能

## 依存チケット
- 003 / 004（WorkSession データ・編集フォーム）

## タスク

### アプリケーション層
- [×] `sessionListProvider` — 全セッションを降順（新しい順）で返す
- [×] 日付グループ化ロジック（同じ日のセッションをまとめる）

### プレゼンテーション層

#### 一覧表示
- [×] 日付ヘッダー付きのグループ化リスト（`SliverList` + sticky header）
- [×] セッションカード
  - [×] カテゴリアイコン・カラー
  - [×] カテゴリ名
  - [×] 作業時間（HH:MM 形式）
  - [×] 金額（円）
  - [×] 入力方式バッジ（タイマー / 手動）
  - [×] メモ（あれば1行表示、省略表示）
- [×] カードタップ → 手動入力フォームへ遷移（編集モード）
- [×] カードの長押し or スワイプ → 削除（確認ダイアログ付き）

#### 手動入力ボタン
- [×] FAB（`+` ボタン）で手動入力フォームを開く

#### 空状態
- [×] セッションが0件の場合、イラスト + メッセージを表示（「まだ記録がありません」）

## 完了条件
- セッション一覧が日付グループ付きで表示される
- 編集・削除が動作する
- 手動入力フォームへ遷移できる
- `flutter analyze` がエラーなし
