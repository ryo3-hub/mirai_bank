# 006 カレンダー表示

## 概要
月間カレンダーで日ごとの累積金額をヒートマップ表示し、日付タップでその日のセッション一覧を表示する。

## 対象フェーズ
Phase 2 — 可視化機能

## 依存チケット
- 001（DB 基盤）
- 002（カテゴリ）
- 003 / 004（WorkSession データ）

## タスク

### アプリケーション層
- [×] `dailyAmountMapProvider(month)` — 指定月の「日付 → 合計金額」マップを返す
- [×] `sessionsOnDayProvider(date)` — 指定日のセッション一覧を返す

### プレゼンテーション層

#### カレンダービュー
- [×] `table_calendar` を使って月間カレンダーを表示
- [×] 各日付セルに合計金額を小さく表示
- [×] 金額に応じたヒートマップ着色（金額が多いほど濃い色）
  - [×] 0円: デフォルト背景
  - [×] 金額に応じて 4 段階程度の色濃度を設定
- [×] 前月 / 次月への矢印ナビゲーション
- [×] 月切替時に dailyAmountMapProvider を再取得

#### 日別セッション一覧
- [×] 日付タップで下部にセッション一覧を展開（`DraggableScrollableSheet` or `BottomSheet`）
- [×] セッションカード（カテゴリ色・アイコン / 時間 / 金額 / メモ）
- [×] セッションカードタップ → 編集フォームへ遷移（004 と連携）

## 完了条件
- 月間カレンダーにヒートマップが表示される
- 日付タップでその日のセッションが表示される
- 前月 / 次月の切替が動作する
- `flutter analyze` がエラーなし
