# 005 ホーム画面 / ダッシュボード

## 概要
累積金額の大きな表示・今日 / 今月サマリ・目標進捗バーを1画面にまとめたダッシュボードを実装する。

## 対象フェーズ
Phase 1 — MVP コア（累積表示・タイマー導線）
Phase 3 — 目標進捗バーは 009_goal_setting.md と連携

## タスク

### アプリケーション層
- [×] `totalAmountProvider(period)` — 全期間 / 今月 / 今週 / 今日の合計金額を返す
- [×] `categoryBreakdownProvider(period)` — カテゴリ別金額の内訳リストを返す

### プレゼンテーション層

#### 累積金額エリア
- [×] 大きな数字で累積金額を表示（アニメーションカウントアップ）
- [×] 期間切替タブ or チップ（全期間 / 今月 / 今週 / 今日）
- [×] 期間切替時にアニメーションで金額が更新される

#### 今日 / 今月サマリ
- [×] 今日の金額・時間の小さなカード表示
- [×] 今月の金額・時間の小さなカード表示

#### タイマー導線
- [×] タイマー未起動時：「学習を開始する」FAB or ボタン → 003 のタイマー UI に遷移
- [×] タイマー計測中：経過時間と金額をインライン表示 + 停止ボタン

#### 目標進捗バー（009 完了後に実装）
- [×] アクティブな目標を最大2件表示
- [×] `LinearProgressIndicator` で達成率を表示
- [×] 達成済みの目標はチェックマーク表示

## 完了条件
- 期間切替で金額が正しく更新される
- タイマー計測中の状態がホーム画面に反映される
- `flutter analyze` がエラーなし
