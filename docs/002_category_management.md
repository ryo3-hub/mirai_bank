# 002 カテゴリ管理機能

## 概要
学習カテゴリの作成・編集・削除と、カテゴリごとの時給・アイコン・カラー設定を実装する。

## 対象フェーズ
Phase 1 — MVP コア

## データモデル（参照）
`Category` エンティティ（id / name / hourlyRate / colorCode / iconCode / createdAt / updatedAt / deletedAt）

## タスク

### ドメイン層
- [×] `Category` エンティティクラスの実装（`lib/features/category/domain/category.dart`）
- [×] カテゴリ名バリデーション（空文字不可・最大文字数）
- [×] 時給バリデーション（1以上の整数）

### インフラ層
- [×] `CategoryRepository` インターフェース定義
- [×] `CategoryRepositoryImpl`（Drift 実装）
  - [×] 全件取得（ソフトデリートを除く）
  - [×] 1件取得
  - [×] 作成（UUID v4 付与、createdAt / updatedAt 自動設定）
  - [×] 更新（updatedAt 自動更新）
  - [×] ソフトデリート（deletedAt に現在時刻をセット）

### アプリケーション層
- [×] `categoriesProvider`（全カテゴリのストリーム）
- [×] `CategoryNotifier`（作成 / 更新 / 削除操作）

### プレゼンテーション層
- [×] カテゴリ一覧画面（設定画面の子ページ）
  - [×] カテゴリカード（アイコン・カラー・名前・時給を表示）
  - [×] 削除スワイプ or 削除ボタン（確認ダイアログ付き）
- [×] カテゴリ作成 / 編集ボトムシート or 画面
  - [×] 名前入力フィールド
  - [×] 時給入力フィールド（数値キーボード）
  - [×] カラーピッカー（プリセット色から選択）
  - [×] アイコン選択（`Icons` からプリセット一覧）
  - [×] 保存 / キャンセルボタン

### 初期データ
- [×] アプリ初回起動時にデフォルトカテゴリを1件作成

## 完了条件
- カテゴリの作成・編集・削除が動作する
- ソフトデリートされたカテゴリが一覧に表示されない
- `flutter analyze` がエラーなし
