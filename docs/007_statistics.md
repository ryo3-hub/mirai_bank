# 007 統計・グラフ画面

## 概要
期間別の累積金額推移グラフとカテゴリ別円グラフを表示する統計画面を実装する。

## 対象フェーズ
Phase 2 — 可視化機能

## 依存チケット
- 001(DB 基盤)
- 002(カテゴリ)
- 003 / 004(WorkSession データ)

## タスク

### アプリケーション層
- [×] `periodSummaryProvider(period)` — 週 / 月 / 年 / 全期間の集計データを返す
  - 日次データ(棒グラフ用): `List<{date, amount}>`
  - カテゴリ別内訳: `List<{category, amount, ratio}>`
- [×] 期間区分ロジック(週 = 直近7日、月 = 当月、年 = 当年、全期間)

### プレゼンテーション層

#### 期間切替
- [×] タブ or チップで「週 / 月 / 年 / 全期間」を切替
- [×] 切替時にグラフがアニメーション更新

#### 推移グラフ(`fl_chart`)
- [×] 棒グラフ(日次 / 週次 / 月次の金額推移)
- [×] X軸ラベル(日付 / 週 / 月を `intl` でフォーマット)
- [×] Y軸ラベル(円単位、K表記対応)
- [×] タップしたバーにツールチップで金額を表示

#### カテゴリ別円グラフ(`fl_chart`)
- [×] `PieChart` でカテゴリ別割合を表示
- [×] カテゴリ色をカテゴリの colorCode に合わせる
- [×] 凡例(カテゴリ名・金額・割合)

#### サマリカード
- [×] 期間合計金額
- [×] 期間合計時間
- [×] 最も学習したカテゴリ

## 完了条件
- 4期間切替でグラフが正しく更新される
- 棒グラフ・円グラフが正しいデータで表示される
- `flutter analyze` がエラーなし
