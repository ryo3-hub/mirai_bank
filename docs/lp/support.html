よくあるご質問

時給は途中で変更できますか？

はい、カテゴリの編集からいつでも変更できます。変更後の新しい記録には新時給が適用されますが、過去の記録の金額は変動しません（記録時に確定する仕様）。

データは復元できますか？

「設定 → データ管理」からエクスポート / インポートができます。機種変更時は、エクスポートで JSON を書き出して、新端末で同じバージョンのアプリでインポートしてください。

通知が来ません

OS の設定でアプリの通知が許可されているか、「設定 → 通知設定」でリマインダーが有効になっているか、曜日・時刻が正しく設定されているかをご確認ください。

タイマー中にアプリを閉じても大丈夫？

はい。タイマーは開始時刻を保存する方式で動いているので、アプリを閉じても、端末をスリープしても計測は継続します。復帰時に経過時間がそのまま反映されます。
