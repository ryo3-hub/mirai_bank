プライバシーポリシー

最終更新日: 2026年5月26日

mirai_bank（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本ポリシーでは、本アプリがどのような情報を取り扱うかを説明します。

1. 取得する情報

本アプリは、以下のデータをユーザーの端末内にローカル保存します。

  • ユーザーが入力したカテゴリ名 / 時給 / 学習時間 / メモ
  • 設定した目標と達成状況
  • リマインダー通知の設定（曜日 / 時刻 / 有効・無効）
  • アプリの利用設定（テーマ、表示設定など）

これらは端末内にのみ保存され、本アプリの開発者を含む第三者には一切送信されません。

2. 個人情報の取得

本アプリは、氏名、メールアドレス、電話番号、住所、生年月日、その他お客様を個人として特定できる情報を一切取得しません。

3. パーミッション（権限）の使用

本アプリは以下のパーミッションを利用します。

  • 通知（ローカル通知）: 学習リマインダーや目標達成のお知らせを端末内で表示するために使用します。OS の通知許可ダイアログで拒否することができます。

4. 第三者サービス

本アプリでは、アプリの安定性向上を目的として、クラッシュレポートサービス Sentry（Functional Software, Inc.、米国）を利用しています。本アプリがクラッシュした場合に、以下の情報が Sentry に送信されます。

  • クラッシュ時のスタックトレース
  • アプリのバージョン / ビルド番号
  • OS の種類とバージョン
  • 端末のモデル名

ユーザーを個人として特定する情報（氏名・メールアドレス・IP アドレス等）は送信されません。Sentry のプライバシーポリシーは Sentry 社の Web サイトをご確認ください。

広告ネットワーク・アナリティクス等の他の第三者サービスは利用していません。将来的に追加する場合は、本ポリシーを更新してお知らせします。

5. 子どもの利用について

本アプリは全年齢を対象としています。13 歳未満のお子様が利用される場合は、保護者の方の同意のもとでご利用ください。

6. データの保存と削除

本アプリの全データは、端末内のローカルストレージ（SQLite データベース）に保存されます。アプリをアンインストールすると、これらのデータもすべて削除されます。

7. データの第三者提供

本アプリは、ユーザーのデータを第三者に提供しません。

8. プライバシーポリシーの変更

本ポリシーは、必要に応じて改訂することがあります。重要な変更がある場合は、アプリ内またはストアのリリースノートで通知します。

9. お問い合わせ

本ポリシーに関するご質問やご意見は、サポートページ よりご連絡ください。

10. 準拠法

本ポリシーは日本国の法律に準拠します。
