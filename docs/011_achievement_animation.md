# 011 達成演出

## 概要
目標達成時のお祝いアニメーションとフィードバック演出を実装する。

## 対象フェーズ
Phase 3 — 継続支援機能

## 依存チケット
- 009（目標達成検知）
- 010（達成通知）

## タスク

### プレゼンテーション層

#### 達成ダイアログ / オーバーレイ
- [×] 目標達成時にフルスクリーンオーバーレイ or ダイアログを表示
  - [×] 紙吹雪アニメーション（`confetti` パッケージ or カスタム実装）
  - [×] 「達成！」テキストのスケールアニメーション
  - [×] 目標名・達成金額の表示
  - [×] 閉じるボタン

#### 連続学習節目演出
- [×] 3日 / 7日 / 30日達成時にバナー or スナックバーを表示
  - [×] ストリークバッジアイコンのアニメーション
  - [×] 「○日連続達成！」メッセージ

#### セッション保存後のフィードバック
- [×] タイマー停止 / 手動記録後に小さなアニメーション（コインが積み上がるなど）
- [×] 保存した金額を一瞬フラッシュ表示

### アプリケーション層
- [×] `achievementQueueProvider` — 未表示の達成イベントをキューで管理
- [×] 画面表示中に達成イベントがあれば演出をトリガーする仕組み

## パッケージ候補
- `confetti`: 紙吹雪アニメーション（`^0.7.0`）

## 完了条件
- 目標達成時に紙吹雪アニメーションが表示される
- 連続学習節目のメッセージが表示される
- セッション保存後の金額フラッシュが動作する
- `flutter analyze` がエラーなし
